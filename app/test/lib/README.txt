This directory includes external testing libs. Namely:

jasmine-1.1.0
  This dir contains the Jasmine library. Downloaded from
  <http://pivotal.github.com/jasmine/>.

sinon-1.3.1.js
 This is the Sinon.JS lib. Sinon.JS is used to create spies, stubs,
 mocks, etc. Downloaded from <http://sinonjs.org>.
    
jasmine-sinon.js
  This script includes a set of custom matchers for using the Sinon.JS
  with Jasmine. Downloaded from <https://github.com/froots/jasmine-sinon>.
